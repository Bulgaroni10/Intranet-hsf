# GSF Hub — Design System

## Nome do produto

GSF Hub  
Plataforma Corporativa do Grupo São Francisco

## Objetivo visual

Transmitir:

- confiança
- tecnologia
- organização
- ambiente hospitalar
- padrão corporativo

## Espaçamento

Baseado em múltiplos de 8px:

- 8px
- 16px
- 24px
- 32px
- 40px
- 48px

## Componentes obrigatórios

Toda página deve seguir:

1. Breadcrumb
2. Page Header
3. KPIs
4. Filtros
5. Conteúdo principal
6. Paginação
7. Ações

## Regra principal

Nenhuma tela deve criar padrão visual próprio.  
Se um componente for reutilizável, ele deve ir para o Core.
